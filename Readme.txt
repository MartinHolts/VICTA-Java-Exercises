Run MainWindow.java file.
Type file name of the level after the main file to run a map.
Like so: java MainWindow level1.skb

Use keys up, down, left, right to move the character around.


COLOR DEFINITIONS

Color White is a path where you can move with the character.
Color Gray is a wall, where you can't move with character or move the boxes to.
Color Orange is boxes that the character can move around.
Color Green is the location the boxes need to be moved to in order to finish the level.
Color red is the character you can move around using arrow keys.
